# Employee Attrition Prediction

An end-to-end machine learning project to predict employee attrition using the IBM HR Analytics dataset. Includes full EDA, preprocessing pipeline, multi-model comparison, hyperparameter tuning, SHAP explainability, and an interactive Streamlit app.

---

## Setup

### 1. Clone the repo
```bash
git clone https://github.com/your-username/employee-attrition-prediction.git
cd employee-attrition-prediction
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Download the dataset
Download `WA_Fn-UseC_-HR-Employee-Attrition.csv` from [Kaggle](https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset) and place it in the `data/` folder.

### 4. Train the model
```bash
python src/train.py
```
This saves the best model to `assets/attrition_model.pkl`.

### 5. Run the Streamlit app
```bash
streamlit run src/app.py
```

## Approach

### EDA
- Class imbalance analysis (~16% attrition rate)
- Attrition rate by categorical features (Department, JobRole, OverTime, MaritalStatus, Gender, BusinessTravel)
- KDE plots for numerical features split by attrition
- Stacked bar charts for ordinal satisfaction features
- Correlation heatmap and top features correlated with attrition
- Age and income group deep dive
- Department × OverTime interaction heatmap

### Preprocessing
- Dropped zero-variance columns (`EmployeeCount`, `Over18`, `StandardHours`, `EmployeeNumber`)
- Label encoded target (`Attrition`: Yes→1, No→0)
- Skewness check on continuous features → Yeo-Johnson transform for |skew| > 0.5
- StandardScaler on normal continuous + ordinal encoded features
- OneHotEncoder (`drop='first'`) on nominal features
- OrdinalEncoder + StandardScaler on `BusinessTravel`
- All preprocessing wrapped in a `ColumnTransformer` inside an `imblearn Pipeline`

### Handling Class Imbalance
- SMOTE applied inside the pipeline (train set only, no leakage)
- `class_weight='balanced'` for sklearn models
- `scale_pos_weight=6` for XGBoost

### Hyperparameter Tuning
- `RandomizedSearchCV` on GradientBoosting with 50 iterations, 5-fold CV, scoring on F1
- Tuned: `n_estimators`, `learning_rate`, `max_depth`, `min_samples_split`, `subsample`, `min_samples_leaf`

### Explainability (SHAP)
- Global feature importance (bar plot)
- Feature impact and direction (beeswarm plot)
- Individual employee explanation (waterfall plot)
- Top driver: **OverTime** — by far the most impactful feature

## Key Business Insights

- **OverTime** is the single strongest predictor of attrition — employees working overtime leave at ~3x the rate of those who don't
- **Low MonthlyIncome** strongly correlates with attrition — the lowest income group leaves at 3-4x the rate of the highest
- **Young employees (18-25)** have the highest attrition rate
- **Single employees** leave significantly more than married or divorced ones
- **Low StockOptionLevel** increases attrition — equity compensation is a strong retention tool
- **Sales Representatives** have the highest attrition rate of all job roles
